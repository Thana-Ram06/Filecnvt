Build a modern, privacy-first file conversion web app similar to tree.je.

## Core Concept
This app allows users to convert files (PDF, images, etc.) directly in the browser without uploading them to any server.

IMPORTANT:
- All processing must happen client-side
- No backend file uploads
- No database
- No authentication

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- PDF processing: pdf-lib or pdfjs
- Image processing: Canvas API
- Fully deployable on Vercel

## Key Principle
"Your files never leave your device"

## Features

### PDF Tools
- PDF to Image (JPG/PNG)
- Image to PDF
- TXT to PDF
- HTML to PDF (basic)

### Image Tools
- JPG to PNG
- PNG to JPG
- JPG to WEBP
- WEBP to PNG

## UI / UX Design (VERY IMPORTANT)

Follow tree.je style:
- Ultra minimal interface
- Centered layout
- No clutter
- No heavy cards or shadows
- Soft borders only
- Smooth transitions

### Layout

#### Homepage (/)
- Centered title (e.g. "Convert files instantly")
- Short tagline (privacy-focused)
- List of tools (simple text buttons or minimal cards)

#### Tool Page (/tools/[tool])
Example: /tools/pdf-to-image

- Centered container
- File upload area (drag & drop)
- Convert button
- Output preview/download
